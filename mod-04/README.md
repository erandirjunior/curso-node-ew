## Uso de recursos no node

Instalando Mocha globalmente:
```bash
npm install -g mocha
```

Instalando Mocha como dependência de desenvolvimento:
```bash
npm i --save-dev mocha
```

Executando testes com mocha:
```js
mocha tests.js
```

Instalando biblioteca **nock** que simula requições:
```bash
npm install nock
```
